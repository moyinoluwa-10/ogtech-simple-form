# OgTech Simple Sign Up Form

A clean and responsive sign-up form built with HTML and CSS as part of an OgTech Course.

## Features

- Simple and user-friendly interface
- Responsive design
- Form fields for:
  - First Name
  - Last Name
  - Password
  - Confirm Password
  - Phone Number
  - Email

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Open Sans, Roboto Slab)

## Usage

Simply open `index.html` in your web browser to view the form.

## Preview

The form includes a modern design with input fields for user registration and a link to sign in for existing users.
